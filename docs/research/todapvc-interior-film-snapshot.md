## 2. Product structure ที่เห็นจาก Category

ภายใต้หัวข้อ `Interior Film (ฟิล์มตกแต่งภายใน)` มีรายการ:

- **TODA Primer – น้ำยารองพื้นสูตรน้ำ**
  - ใช้งานร่วมกับ Interior Film
  - น้ำยารองพื้นสำหรับเคลือบพื้นผิว เพื่อเพิ่มการยึดเกาะของกาว → แสดงว่า TODA มี **own primer** และมอง value chain แบบครบ (ไม่ใช่ขายแต่ฟิล์ม)

- **Interior Film Series (ความหนา/ลักยณะ)**
  - Reflex Glossy 0.45mm
  - Fabric 0.30mm
  - Concrete 0.30mm
  - Marble 0.30mm
  - Metal 0.30mm
  - Premium Wood 0.30mm
  - Soft Matt 0.30mm
  - Basic Solid 0.30mm

รายละเอียดที่เห็น:
- หน่วยขาย: `เมตร` พร้อม code เช่น `Reflex Glossy 927-01`, `Fabric 926-01` ฯลฯ
- ระบุว่า "(มีหลายคุณสมบัติให้เลือก)" → น่าจะมีหลายสี/ลายภายใต้แต่ละ series
- ความหนา 0.30mm เป็น base สำหรับหลาย series (wood/solid/matt/metal/etc.)

**Takeaway:**
- TODA แบ่ง product structure ตาม **surface finish / pattern family + thickness** คล้าย Younglim/XSurface แต่ใช้ naming ตัวเอง
- มี product line ครบตั้งแต่ **Basic Solid → Soft Matt → Premium Wood → Marble/Concrete/Metal → Reflex Glossy** + primer

## 4. TODA PVC บนเรดาร์ของ Zenta

### 4.1 เขาอยู่ในกลุ่มไหนของ landscape?

จาก category เดียว สันนิษฐานได้ว่า TODA อยู่ในกลุ่ม:
- **Local brand / supplier ที่มี interior film เป็น product line สำคัญ**
- ผสมบทบาท **material supplier + tool (primer)** → value chain คล้าย one-stop ภายใน product line ของตัวเอง

### 4.2 สิ่งที่น่าสนใจสำหรับ Zenta

- **Product taxonomy**
  - การแบ่งลำดับ series (Basic Solid → Soft Matt → Premium Wood → Marble/Concrete/Metal → Reflex Glossy) เป็นตัวอย่างโครง product tree ที่เข้าใจง่ายสำหรับคนไทย
  - Zenta สามารถใช้เป็น reference ในการ design series/collection ของตัวเอง (จะเหมือน/ต่างยังไง)

- **Primer / system thinking**
  - TODA มี primer ของตัวเอง → ขาย "ระบบ" มากกว่าตัวฟิล์มอย่างเดียว
  - Zenta อาจคิดทั้งมุม UX (แนะนำสินค้าเสริม/primer/tools ใน product detail) และมุม business (bundle solution)

### 4.3 ระดับ priority

- TODA ยังไม่ได้ถูกวิเคราะห์ลึกแบบ Hyundaifilm/Goodfilm/Younglim/XSurface เพราะเรายังเห็นแค่หน้า category เดียว
- แต่ควรถือว่าเป็น **"Tier-2 competitor on radar"**:
  - มี product tree ค่อนข้างชัด
  - เคย approach บริษัทคุณโดยตรง (insight offline สำคัญ)

หากในอนาคต Zenta ต้องตัดสินใจด้าน series naming, pricing หรือ partnership กับ local supplier/brand อื่น ๆ สามารถกลับมาเจาะ TODA เพิ่มได้จากไฟล์ snapshot นี้และจากหน้าอื่น ๆ บนเว็บ TODA
